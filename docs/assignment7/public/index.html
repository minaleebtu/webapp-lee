BTU Music Club App

A Web App built with Plain JS and Firebase

This app supports the following operations:

  • Create a new ensemble record
  • Retrieve and list all ensembles
  • Update an ensemble record
  • Delete an ensemble record
  • Create a new event record
  • Retrieve and list all events
  • Update an event record
  • Delete an event record
