A Minimal effort Web App built with Plain JS and Firebase

This app, which is accompanied by a tutorial, is the 1st example app of our incremental sequence of six examples of Plain JavaScript and Firebase Web Apps, which are also discussed in the acompanying book Building Web Apps with Plain JS and Firebase. This app supports the following operations:

  • Create a new book record
  • Retrieve and list all book records
  • Update a book record
  • Delete a book record
